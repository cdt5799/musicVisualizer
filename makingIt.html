Team Members:

  1. Chelsea Triebwasser - Chelsea's Page

Abstract

We are creating a HTML5 Media Experience that relates sound and image. We want to create a visualizer effect so when you upload a song the page generated animated imagery based on the song and the notes in the song.

Technical and Design Decisions

The major technical decision was based on what Google Chrome allowed us to do with the frequences and how they were read in.

The major design decisions were deciding on what shapes to use and how to use them. Also another design decision included what we wanted the user to be able to control.

Extras

We worked really hard on this project using extensive JavaScript and canvas, as well as css to make it visually apealing.

We also incorporated a drag and drop feature so the user could pick the song they wanted to visualize. For user interactivity we incorporated many different effects so the user could create their own visual experience.
